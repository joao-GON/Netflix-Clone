## 🚀 Visão Geral

Este projeto simula o fluxo principal de uma plataforma de streaming:

* Login de usuário
* Seleção de perfil
* Navegação para catálogo de filmes

Tudo isso com uma interface responsiva, sistema de tema (Dark/Light Mode) e persistência de estado — proporcionando uma experiência fluida e próxima de um produto real.

---

## 🧩 Estrutura do Projeto

```
📁 projeto
├── login.html        # Tela de login
├── index.html        # Seleção de perfis
├── catalogue.html    # Catálogo de filmes
├── style.css         # Estilização global e responsiva
├── script.js         # Controle de tema e interações
└── navigation.js     # Navegação e gerenciamento de estado
```

---

## 🔄 Fluxo da Aplicação

1. Usuário acessa a tela de login
2. Insere credenciais (simulação)
3. É redirecionado para seleção de perfil
4. Escolhe um perfil
5. Acessa o catálogo de filmes
6. Pode alternar tema ou realizar logout
